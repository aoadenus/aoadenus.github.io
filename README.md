## Local Preview

No build step needed. Just open any `.html` file in your browser, or run a simple local server:

```bash
# Python 3
python -m http.server 8000
# Then visit http://localhost:8000
```

---

## Updating Content

All content is in the HTML files. Common updates:

- **New project** → add a `.pcard` block in `projects.html`
- **New job** → add a `.tl-entry` block in `experience.html`
- **New skill** → add a `.chip` span in `skills.html`
- **Resume** → replace `Adetutu_Adenusi_Resume.pdf` (keep the same filename)
